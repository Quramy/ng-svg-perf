# ng-svg-perf

[![Greenkeeper badge](https://badges.greenkeeper.io/Quramy/ng-svg-perf.svg)](https://greenkeeper.io/)

Measures SVG rendering performance with Angular 1 and 2. It uses D3's force layout.
